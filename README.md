# Family Files Storage App

## Project Overview
The **Family Files Storage App** is a web application designed to help users manage and store information about family members and important files. It provides a user-friendly interface to add family members, input their details, and upload files for secure storage. The app utilizes local storage for data persistence, making it ideal for personal use without the need for a backend server.

## Installation
To run the Family Files Storage App, follow these steps:

1. **Download the HTML File**:
   - Download the `family-files-storage-app.html` file from the repository.

2. **Open in a Browser**:
   - Open the downloaded HTML file in any modern web browser (e.g., Chrome, Firefox, Edge).

No additional setup or dependencies are required.

## Usage
Once the app is opened in the browser, you can:

1. **Add Family Members**:
   - Fill out the family member form by providing their name, birthdate, age, relation, and contact numbers.
   - Click on "Add Member" to submit the data, which will then be displayed in the family members list.

2. **Upload Files**:
   - Use the file upload section to store important files. Provide a name for the file and select the file to upload.
   - Click on "Upload File" to add the file to the storage list.

3. **View Family Members and Files**:
   - View the list of added family members and uploaded files on the main page.
   - You can delete any family member or file by clicking the delete button next to the respective entry.

## Features
- **User-Friendly Interface**: Designed with a responsive layout using Tailwind CSS for aesthetic appeal and usability.
- **Family Member Management**: Add, display, and delete family members with their relevant information.
- **File Storage**: Upload and display important files securely.
- **Local Storage**: Data is stored in the browser's local storage, ensuring persistence across page refreshes.
- **Mobile Friendly**: The app is optimized for use on mobile devices.

## Dependencies
This project utilizes the following external libraries:
- **Tailwind CSS**: For styling the application (loaded from CDN).
- **Font Awesome**: For icons used throughout the application (loaded from CDN).
